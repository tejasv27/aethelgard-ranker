# Executive Summary: Aethelgard
Aethelgard is a high-performance, hybrid candidate screening engine designed to solve the critical problem of "resume inflation." By employing a deterministic O(N) streaming algorithm combined with advanced Cross-Encoder semantic models and LLM-driven dynamic weighting, Aethelgard identifies genuine engineering talent hidden within massive candidate pools. 

## Key Innovations
- **Scalability:** Processes 100,000+ records in seconds without memory overflow.
- **Anti-Cheat:** Honeypot detection penalizes keyword-stuffing.
- **Deep Alignment:** Sliding-window cross-encoders bypass standard token limits.
- **Adaptive Learning:** SQLite-backed RLRF permanently learns from recruiter actions.
